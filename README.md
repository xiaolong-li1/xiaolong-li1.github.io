## 如何使用✊
1. 将本仓库down 下来 ✔️
2. 修改`index.html`中，下方`data`中的配置数据✔️
```js
// 个人信息部分
header: {
    name: 'nickname',
    imgUrl: '头像路径',
    signature: '简短介绍 or  座右铭  or  口号'
    },
// 站点信息部分
items: [
    {
        href: '目标地址',// 站点所在地址
        icon: '图标', // 要使用的图标，请自行引入阿里矢量库
        des: '描述' // 描述文字
    },
]  
```

3. 将下方 `a` 标签中的 `备案号：123456789 ` 替换为你自己的

4. 将修改好的文件上传到你的服务器上就行啦~✔️
